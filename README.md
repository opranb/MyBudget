# MyBudget 💰

**MyBudget** — это простое десктопное приложение для учёта личных финансов, написанное на Python с использованием PyQt6 и SQLite. Оно позволяет добавлять доходы и расходы, удалять транзакции, отслеживать текущий баланс и визуализировать динамику последних 10 операций на графике.

![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)
![PyQt6](https://img.shields.io/badge/PyQt6-6.0+-green.svg)

## 📸 Интерфейс

Приложение имеет тёмную тему с акцентным фиолетовым цветом:
- Поле ввода суммы с валидацией (только числа и точка)
- Поле для описания транзакции
- Поле для ID транзакции при удалении
- Таблица со всеми операциями (дата, описание, сумма, ID)
- Баланс отображается зелёным (положительный) или красным (отрицательный)

## ✨ Возможности

-  Добавление доходов и расходов
-  Удаление транзакций по ID
-  Автоматический пересчёт баланса
-  График динамики последних 10 транзакций (matplotlib)
-  Современный тёмный интерфейс
-  Автоматическая нумерация ID без пропусков (при удалении ID переиспользуются)
-  Локальное хранение данных в SQLite

## 🛠 Технологии

- **Python 3.8+**
- **PyQt6** — графический интерфейс
- **SQLite3** — база данных
- **Matplotlib** — построение графиков

## 📦 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/opranb/mybudget.git
cd mybudget
```

### 2. Установить зависимости

```bash
pip install PyQt6 matplotlib
```

### 3. Запустить приложение

```bash
python MyBudget.py
```
